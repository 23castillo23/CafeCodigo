PARTE 1 — El vocabulario esencial
Antes de escribir código necesitas conocer exactamente qué significa cada palabra

Muchos principiantes confunden estos términos y eso genera errores difíciles de encontrar. Apréndetelos bien desde el inicio.

¿Qué es una Etiqueta (Tag)?

Una etiqueta es una palabra clave rodeada de los signos < y >. Le dice al navegador qué tipo de contenido viene a continuación.

<h1> Etiqueta de apertura → 'empieza un título grande'
</h1> Etiqueta de cierre → 'aquí termina el título grande'

🍕 Analogía: los letreros de sección en el supermercado

Piensa en las etiquetas como letreros que pones al inicio y al final de algo. En un supermercado, hay un letrero que dice 'FRUTAS' cuando empiezas esa sección y otro que indica donde termina. Las etiquetas hacen exactamente lo mismo: marcan el inicio y el final de cada tipo de contenido.

¿Qué es un Elemento?

Un elemento es el paquete completo: etiqueta de apertura + contenido + etiqueta de cierre. Es la unidad fundamental del HTML.

<h1>Tacos de Frijoles</h1>
      Apertura            Contenido      Cierre
                           Todo junto = 1 ELEMENTO

¿Qué es un Atributo?

Un atributo es información extra que se pone dentro de la etiqueta de apertura para configurar cómo funciona el elemento. Siempre tiene el formato nombre="valor".

<img src="tacos.jpg" alt="Foto de mis tacos" width="400">
        src = ruta de la imagen alt = descripción width = ancho en px

🍕 Analogía: los botones de un microondas

Los atributos son como los botones y perillas de un aparato electrodoméstico. El microondas (elemento) tiene botones de: tiempo, potencia, función. Cada botón (atributo) tiene un valor que configura el aparato.

¿Qué es una Clase (class)?

Una clase es un nombre que tú inventas y que puedes ponerle a uno o varios elementos. Sirve para que el CSS los identifique y les aplique estilos.

HTML<p class="ingrediente">1 lata de frijoles negros</p> <p class="ingrediente">2 cucharadas de aceite</p> <p class="ingrediente">1 cebolla pequeña</p> /* En el CSS: */ .ingrediente { color: #374151; }

🍕 Analogía: el uniforme del equipo

La clase es como el uniforme de un equipo de fútbol. Todos los jugadores del mismo equipo llevan el mismo uniforme (clase). No importa si son delanteros o defensas, si tienen la misma clase, reciben el mismo estilo.

¿Qué es un ID?

Un id es un identificador único. Solo puede existir UN elemento con cada id en toda la página. Se usa para identificar elementos específicos.

HTML<h1 id="titulo-principal">Tacos de Frijoles</h1> /* En el CSS: */ #titulo-principal { font-size: 48px; }

🍕 Analogía: tu CURP o número de cédula

El ID es como tu número de CURP: es único en todo el país. No puede haber dos personas con el mismo número. Las clases son como un apellido: muchas personas pueden compartirlo.

PARTE 2 — Crea tus archivos
La forma correcta de organizar un proyecto web desde el principio

Vamos a crear 2 archivos en la misma carpeta.

  • Crea una carpeta llamada receta-tacos en tu escritorio o en Documentos
  • Dentro, crea el archivo index.html (el HTML)
  • En la misma carpeta, crea el archivo styles.css (el CSS)

💡 ¿Por qué se llama index.html?

Cuando escribes una dirección web como www.google.com, el servidor busca automáticamente un archivo llamado index.html para mostrar como página de inicio. Es la convención universal. Siempre nombra tu página principal index.html.

💡 ¿Por qué 2 archivos separados?

Es la mejor práctica profesional. El HTML solo tiene la estructura (el contenido), y el CSS solo tiene los estilos (la apariencia). Todos los desarrolladores profesionales separan HTML y CSS.

PARTE 3 — Escribe el HTML
Lee cada bloque con su explicación antes de escribirlo

BLOQUE 1 — La estructura base obligatoria

Todo archivo HTML válido debe empezar exactamente igual. Estas líneas son el esqueleto que el navegador necesita para entender el documento.

HTML<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mi Receta de Tacos</title> <link rel="stylesheet" href="styles.css"> </head> <body> <!-- Aquí va el contenido visible --> </body> </html>
Línea de código¿Qué hace?
<!DOCTYPE html>Le dice al navegador: 'esto es HTML5, la versión moderna'. Siempre va en la primera línea.
<html lang="es">El elemento raíz. Todo el documento va dentro. lang='es' indica que la página está en español.
<head>La 'cabeza' del documento. El navegador la lee pero no la muestra en pantalla.
<meta charset="UTF-8">Permite usar tildes (á, é, í), la ñ y emojis sin caracteres extraños. Siempre primero dentro del head.
<meta name="viewport">Hace que la página se vea bien en celulares. Sin esto, en móvil el texto se ve muy pequeño.
<title>El texto que aparece en la pestaña del navegador y en los resultados de Google.
<link rel="stylesheet">Conecta el archivo CSS con el HTML. Sin esta línea, los estilos no se aplican.
<body>El 'cuerpo'. AQUÍ va todo el contenido visible: textos, imágenes, listas, etc.
<!-- comentario -->Un comentario. El navegador lo ignora completamente. Úsalo para hacer notas en tu código.

BLOQUE 2 — El encabezado de la página

Escribe esto dentro del <body>. Vamos a crear el encabezado principal de la receta.

HTML<!-- ENCABEZADO DE LA PÁGINA --> <header id="encabezado"> <h1 id="titulo-principal">🌮 Tacos de Frijoles</h1> <p>Una receta mexicana tradicional, sencilla y deliciosa.</p> <p><span class="tiempo">⏱ Tiempo:</span> 30 minutos</p> <p><span class="tiempo">👥 Porciones:</span> 4 personas</p> </header>
EtiquetaDefinición y uso
<header>Etiqueta semántica que marca el encabezado de la página. Google y los lectores de pantalla la reconocen como 'el encabezado del sitio'.
id="encabezado"Atributo que le da a este elemento un identificador único. Lo usaremos en el CSS para aplicarle estilos específicos.
<h1>El título más importante de la página. Debe haber UNO solo por página. Es el 'titular' del periódico.
<p>Párrafo de texto. Crea espacio arriba y abajo automáticamente. Es un elemento de bloque.
<span>Contenedor en línea sin apariencia propia. Se usa para rodear una parte del texto y darle estilos diferentes.
class="tiempo"Le asignamos la clase 'tiempo' a ambos spans. En el CSS los estilizamos juntos con .tiempo { }

BLOQUE 3 — La imagen de la receta

Agrega esto después del </header>. Usamos una URL de internet para que no necesites tener la foto guardada.

HTML<!-- IMAGEN DE LA RECETA --> <section id="imagen-seccion"> <img src="https://placehold.co/600x350/FDE68A/92400E?text=Tacos+de+Frijoles" alt="Foto de tacos de frijoles" width="600" height="350" > </section>

📸 Explicación de la imagen

  • src = la ruta o URL de la imagen. Aquí usamos placehold.co que genera imágenes de prueba.
  • alt = texto alternativo. Si la imagen no carga, el usuario ve este texto. NUNCA lo omitas.
  • width / height = ancho y alto en píxeles. Evitan que la página 'salte' mientras carga la imagen.
  • <section> = etiqueta semántica que agrupa contenido relacionado. Como un capítulo de un libro.

BLOQUE 4 — Los ingredientes (lista sin orden)

Los ingredientes van en una lista sin orden porque no importa en qué secuencia los listas.

HTML<!-- INGREDIENTES --> <section id="ingredientes"> <h2>🧂 Ingredientes</h2> <p>Para preparar esta receta necesitas:</p> <ul> <li class="ingrediente">1 lata (400g) de frijoles negros cocidos</li> <li class="ingrediente">8 tortillas de maíz</li> <li class="ingrediente">1 cebolla pequeña</li> <li class="ingrediente">2 dientes de ajo</li> <li class="ingrediente">2 cucharadas de aceite vegetal</li> <li class="ingrediente">Sal y pimienta al gusto</li> <li class="ingrediente destacado">1 aguacate (opcional pero recomendado 🥑)</li> </ul> </section>
EtiquetaDefinición y uso
<h2>Subtítulo de segundo nivel. Debajo del h1 en importancia. Puedes tener varios h2 en la misma página.
<ul>Unordered List = Lista Sin Orden. Se usa cuando el orden de los elementos NO importa.
<li>List Item = Elemento de lista. Siempre va dentro de <ul> o <ol>.
class="ingrediente"Clase que le ponemos a cada li. Así damos el mismo estilo a todos con: .ingrediente { }
class="ingrediente destacado"Este li tiene DOS clases: 'ingrediente' y 'destacado'. Recibirá los estilos de ambas clases.

BLOQUE 5 — Los pasos (lista con orden)

Los pasos de preparación van en lista ordenada porque el orden sí importa.

HTML<!-- PASOS DE PREPARACIÓN --> <section id="preparacion"> <h2>👩‍🍳 Preparación</h2> <ol> <li>Pica finamente la cebolla y el ajo.</li> <li>Calienta el aceite en un sartén a fuego medio.</li> <li>Sofríe la cebolla y el ajo por 3 minutos.</li> <li>Agrega los frijoles escurridos. Sazona con sal y pimienta.</li> <li>Aplasta los frijoles hasta obtener una pasta. Cocina 5 minutos más a fuego bajo.</li> <li>Calienta las tortillas y sirve. ¡Listo!</li> </ol> </section>
EtiquetaDefinición y uso
<ol>Ordered List = Lista Con Orden. Los elementos se numeran automáticamente (1, 2, 3...). Se usa cuando el ORDEN importa.
<strong>Marca texto como muy importante. Se ve en negritas. Tiene significado semántico.
<em>Marca texto con énfasis. Se ve en cursiva/itálica. Como cambiar el tono de voz al hablar.

🔗 La etiqueta <a> — el hipervínculo

La etiqueta <a> (de 'anchor' = ancla) crea los hipervínculos. href = la URL a donde lleva el enlace. target="_blank" = abre en pestaña nueva. rel="noopener" = medida de seguridad que siempre debes añadir al usar target='_blank'. <footer> = pie de página semántico.

PARTE 4 — Escribe el CSS
Anatomía de una regla CSS y todos los estilos de la receta

Anatomía de una regla CSS

selector {
  propiedad: valor;
  propiedad: valor;
}
TérminoEjemplo¿Qué es?
Selectorh1 / .ingrediente / #titulo¿A quién le aplico el estilo?
Propiedadcolor / font-size¿Qué aspecto quiero cambiar?
Valorred / 36px / bold¿Cómo quiero que quede?
Declaracióncolor: red;Una propiedad + su valor. Siempre termina con ;
Reglah1 { color: red; }El bloque completo: selector + declaraciones entre llaves.

CSS — PARTE 1: Reset y estilos generales

CSS/* Reset: quita márgenes y rellenos por defecto */ * { margin: 0; padding: 0; box-sizing: border-box; } /* BODY: estilos para toda la página */ body { font-family: Arial, Helvetica, sans-serif; font-size: 16px; color: #333333; background-color: #FFF8F0; line-height: 1.6; }
Propiedad CSS¿Qué hace?
* { }El selector * (asterisco) aplica a todos los elementos de la página.
margin: 0;Quita el espacio exterior de todos los elementos.
padding: 0;Quita el espacio interior. Junto con margin:0, deja todo 'limpio'.
box-sizing: border-box;Hace que el padding no aumente el tamaño del elemento.
font-family:Define la fuente. Siempre se ponen varias de respaldo separadas por comas.
color: #333333;Color del texto en hexadecimal. #333333 es un gris oscuro (más suave que negro puro).
background-color: #FFF8F0;Color de fondo. #FFF8F0 es un blanco cálido (ligeramente crema).
line-height: 1.6;El interlineado. 1.6 = 1.6 veces el tamaño de la letra. Ideal para lectura cómoda.

CSS — PARTE 2: Encabezado y título

CSS#encabezado { background-color: #92400E; color: white; text-align: center; padding: 40px 20px; } #titulo-principal { font-size: 48px; font-weight: bold; letter-spacing: 2px; text-transform: uppercase; margin-bottom: 10px; } #encabezado p { font-size: 18px; font-style: italic; margin-top: 8px; } .tiempo { font-weight: bold; color: #FDE68A; text-decoration: underline; }
PropiedadExplicación
#encabezado { }El # indica que es un selector de ID. Solo aplica al elemento con id='encabezado'.
text-align: center;Centra el texto horizontalmente. Valores: left, center, right, justify.
padding: 40px 20px;40px arriba/abajo y 20px izquierda/derecha. Separa el contenido del borde.
letter-spacing: 2px;Espacio entre letras. Con 2px las letras tienen más 'aire', se ve más elegante.
text-transform: uppercase;Transforma el texto a MAYÚSCULAS sin reescribir el HTML.
.tiempo { }El punto indica selector de clase. Aplica a todos los elementos con class='tiempo'.
text-decoration: underline;Subraya el texto. Otros valores: none, line-through, overline.

CSS — PARTE 3: Imagen, listas y footer

CSS#imagen-seccion { text-align: center; padding: 30px; background-color: #FEF3C7; } #imagen-seccion img { border: 5px solid #D97706; border-radius: 12px; } #ingredientes, #preparacion, #datos { padding: 30px 40px; max-width: 700px; margin: 0 auto; } h2 { font-size: 28px; color: #92400E; border-bottom: 3px solid #D97706; padding-bottom: 8px; margin-bottom: 16px; } .ingrediente { font-size: 17px; color: #374151; margin-bottom: 8px; line-height: 1.8; } .destacado { font-weight: bold; color: #065F46; font-style: italic; } a { color: #4F46E5; text-decoration: none; font-weight: bold; } a:hover { text-decoration: underline; color: #3730A3; } footer { background-color: #92400E; color: white; text-align: center; padding: 20px; font-size: 14px; margin-top: 40px; }
PropiedadExplicación
border:El borde visible de un elemento. Formato: grosor estilo color. Ej: 5px solid #D97706.
border-radius: 12px;Redondea las esquinas del elemento. Con 50% se hace un círculo perfecto.
max-width: 700px;El elemento no puede ser más ancho que 700px. Evita que el texto se estire demasiado.
margin: 0 auto;El truco más útil de CSS: centra horizontalmente un elemento con ancho definido.
border-bottom:Solo el borde inferior. Aquí se usa para hacer una línea decorativa debajo del h2.
#ingredientes, #preparacionSelector múltiple: aplica la misma regla a varios selectores separados por coma.
a:hover { }Pseudo-clase: aplica estilos solo cuando el usuario pasa el ratón encima del enlace.

🌮 Tacos de Frijoles

Una receta mexicana tradicional, sencilla y deliciosa.

⏱ Tiempo: 30 minutos 👥 Porciones: 4 personas

🧂 Ingredientes

Para preparar esta receta necesitas:

  • 1 lata (400g) de frijoles negros cocidos
  • 8 tortillas de maíz
  • 1 cebolla pequeña
  • 2 dientes de ajo
  • 2 cucharadas de aceite vegetal
  • Sal y pimienta al gusto
  • 1 aguacate (opcional pero recomendado 🥑)

👩‍🍳 Preparación

  1. Pica finamente la cebolla y el ajo.
  2. Calienta el aceite en un sartén a fuego medio.
  3. Sofríe la cebolla y el ajo por 3 minutos.
  4. Agrega los frijoles escurridos. Sazona con sal y pimienta.
  5. Aplasta los frijoles y cocina 5 minutos más a fuego bajo.
  6. Calienta las tortillas y sirve. ¡Listo!
