Seleccionar y modificar un elemento:

const el = document.querySelector("#miId") el.textContent = "Nuevo texto" // querySelector acepta cualquier selector CSS // #id → por id // .clase → por clase // p → por etiqueta

✏️ Escribe tu código:

💡 El id del párrafo es demo1-texto.
Selecciónalo con: document.querySelector('#demo1-texto')
El # significa "buscar por id" — igual que en CSS.

Cambiar estilos — camelCase:

const el = document.querySelector("#elId") el.style.backgroundColor = "blue" ← background-color el.style.fontSize = "20px" ← font-size el.style.color = "white" el.textContent = "Nuevo"

✏️ Escribe tu código:

💡 El id de la caja es demo2-caja.
En JavaScript las propiedades CSS se escriben en camelCase: backgroundColor no background-color.
Los colores van entre comillas: '#2a6496'

    Eliminar elementos del DOM

    📌 Cada elemento de la lista tiene un botón ✕. Haz que al hacer clic en ese botón, se elimine ese elemento completo.

    Zona de resultado — haz clic en ✕ para eliminar:

    • Aprender HTML
    • Aprender CSS
    • Aprender JavaScript
    • Hacer un proyecto

    Eliminar un elemento:

    // El elemento se elimina a sí mismo elemento.remove() // Desde un hijo, eliminar el padre: boton.addEventListener("click", (e) => { e.target.parentElement.remove() })

    ✏️ Escribe tu código:

    💡 e.target es el botón que se hizo clic.
    e.target.parentElement es el <li> que contiene ese botón.
    .remove() lo elimina completamente de la página.
