Bucle for básico

📌 Usa un bucle for para mostrar los números del 1 al 10, uno por línea.

Estructura del for:

for (let i = 0; i < 5; i++) { console.log(i) ← 0,1,2,3,4 } // inicio ; condición ; avance

Escribe tu código:

💡 Empieza en 1 (let i = 1) y termina en 10 (i <= 10).
Cada vuelta i++ suma 1.
Prueba mostrando solo los pares: if (i % 2 === 0)
2

Recorrer un arreglo con for

📌 Tienes el arreglo ['Lunes','Martes','Miércoles','Jueves','Viernes']. Recórrelo con un bucle for y muestra cada día con su número: "1. Lunes", "2. Martes"...

Acceder a arreglos:

const frutas = ["manzana", "naranja"] for (let i = 0; i < frutas.length; i++) { console.log(frutas[i]) ← cada elemento }

Escribe tu código:

💡 frutas.length da el total de elementos.
frutas[i] accede al elemento en la posición i.
El índice empieza en 0, por eso el número visible es i+1.
3

forEach — la forma moderna

📌 Tienes el arreglo de precios [120, 450, 80, 299, 15]. Usa forEach para mostrar cada precio con formato: "Precio: $120".

forEach recorre todos los elementos:

const nums = [1,2,3] nums.forEach(num => { console.log(num) }) // más limpio que el for clásico

Escribe tu código:

💡 forEach recibe una arrow function que se ejecuta por cada elemento.
El parámetro (precio) toma el valor de cada elemento en cada vuelta.
4

map — transformar cada elemento

📌 Tienes [1, 2, 3, 4, 5]. Usa map para crear un nuevo arreglo donde cada número esté al cuadrado. Muestra el resultado.

map crea un NUEVO arreglo transformado:

const nums = [1,2,3] const dobles = nums.map(n => n * 2) console.log(dobles) ← [2,4,6] // nums no cambia — map crea uno nuevo

Escribe tu código:

💡 map(n => n * n) devuelve el cuadrado de cada elemento.
El arreglo original no cambia — map siempre crea uno nuevo.
Resultado esperado: [1, 4, 9, 16, 25]
5

filter — quedarse con los que cumplen

📌 Tienes [3, 17, 8, 42, 5, 29, 11, 6]. Usa filter para crear un arreglo solo con los números mayores de 10. Muéstralo.

filter conserva solo los que pasan la condición:

const nums = [1,5,12,3,8] const altos = nums.filter(n => n > 6) console.log(altos) ← [12,8]

Escribe tu código:

💡 filter(n => n > 10) devuelve solo los mayores de 10.
Resultado esperado: [17, 42, 29, 11]
Prueba también filtrar los pares: n % 2 === 0
6

reduce — acumular en un valor

📌 Tienes [10, 25, 5, 40, 20]. Usa reduce para calcular la suma total. Muestra: "Total: 100".

reduce acumula todos los elementos en uno:

const nums = [1,2,3,4] const total = nums.reduce((acum, n) => acum + n, 0) console.log(total) ← 10 // acum empieza en 0, va sumando cada n

Escribe tu código:

💡 El 0 al final es el valor inicial del acumulador.
En cada vuelta: acum tiene la suma hasta ahora, n es el elemento actual.
También puedes buscar el máximo: reduce((max, n) => n > max ? n : max, 0)
7

Combinar map + filter

📌 Tienes esta lista de productos:
[{nombre:'Libro', precio:150}, {nombre:'Curso', precio:800}, {nombre:'Lapiz', precio:20}, {nombre:'Monitor', precio:3500}]
Filtra los que cuestan más de 100 y crea un arreglo con solo sus nombres.

Escribe tu código:

💡 Primero filter para quedarte con los caros, luego map para extraer solo el nombre.
O en una línea: productos.filter(p => p.precio > 100).map(p => p.nombre)
Resultado esperado: ['Libro', 'Curso', 'Monitor']
8

🏆 Reto final — Análisis de notas

📌 Tienes el arreglo [85, 42, 91, 67, 78, 55, 93, 38].
Usando forEach, filter, map y reduce calcula y muestra:
• Cuántos aprobaron (mayor o igual a 60)
• El promedio de TODAS las notas
• La nota más alta

Escribe tu código:

💡 Aprobados: filter(n => n >= 60).length
Promedio: suma total dividida entre cantidad de elementos
Máximo: reduce((max, n) => n > max ? n : max, 0)
O más simple: Math.max(...notas)
