JavaScript · Nivel 03

Funciones — Las recetas del código

Crea funciones propias, usa parámetros, devuelve resultados con return y escribe arrow functions. Las funciones son la herramienta más usada en cualquier proyecto.

📝 8 ejercicios⏱️ ~30 minutos📈 Nivel: Principiante-Intermedio

0 de 8 ejercicios completados

1

Tu primera función

📌 Crea una función llamada saludar que muestre '¡Hola! Bienvenido al nivel 03' cuando la llames. Llámala dos veces.

Estructura básica:

function nombre() { // instrucciones } nombre() ← así se llama

✏️ Escribe tu código:

💡 Dentro de las llaves va el console.log().
Para llamarla: saludar() — nombre + paréntesis.
2

Función con parámetro

📌 Crea una función saludarPersona que reciba un nombre y muestre 'Hola, [nombre]!'. Llámala con tres nombres diferentes.

Parámetros:

function saludar(nombre) { console.log(`Hola, ${nombre}!`) } saludar("Ana")

✏️ Escribe tu código:

💡 El parámetro es una variable temporal solo dentro de la función.
Usa template literal: `Hola, ${nombre}!`
3

Función con return

📌 Crea una función sumar que reciba dos números y devuelva su suma. Guarda el resultado en una variable y muéstralo.

return devuelve el resultado:

function doblar(n) { return n * 2 ← devuelve } const r = doblar(5) console.log(r) ← 10

✏️ Escribe tu código:

💡 return a + b
La diferencia: console.log muestra, return devuelve el valor para usarlo después.
4

Varios parámetros

📌 Crea una función calcularArea que reciba base y altura y devuelva el área (base × altura). Pruébala con base=10, altura=5.

✏️ Escribe tu código:

💡 Área de rectángulo: return base * altura
¿Qué pasa si una medida es 0? ¿Y si es negativa?
5

Arrow functions

📌 Reescribe como arrow functions:
• function cuadrado(n) { return n * n }
• function esMayor(a, b) { return a > b }

De función normal a arrow function:

// Normal function sumar(a,b) { return a+b } // Arrow — mismo resultado const sumar = (a,b) => a+b // Un parámetro — sin paréntesis const doblar = n => n*2

✏️ Escribe tu código:

💡 const cuadrado = n => n * n
const esMayor = (a, b) => a > b
Cuando hay un solo parámetro no necesita paréntesis.
6

Función que llama a otra

📌 Crea dos funciones:
• celsiusAFahrenheit(c) — fórmula: c × 9/5 + 32
• describir(temp) — llama a la anterior y muestra: '20°C = 68°F'

✏️ Escribe tu código:

💡 Fórmula: return c * 9/5 + 32
0°C=32°F · 100°C=212°F · 20°C=68°F
Usa .toFixed(1) para un decimal.
7

Parámetros por defecto

📌 Crea una función presentar que reciba nombre y rol (por defecto 'estudiante'). Llámala con y sin el segundo parámetro.

Valores por defecto:

function saludar(nombre, idioma = "español") { console.log(`Hola en ${idioma}, ${nombre}`) } saludar("Ana") ← usa "español" saludar("Ana", "inglés") ← usa "inglés"

✏️ Escribe tu código:

💡 El valor por defecto se usa solo cuando no pasas ese argumento.
Mensaje ejemplo: `${nombre} es ${rol}`
8

🏆 Reto final — Calculadora

📌 Crea 4 arrow functions: sumar, restar, multiplicar, dividir.
Luego una función calcular(a, op, b) que llame a la correcta y muestre: '10 + 5 = 15'.

✏️ Escribe tu código:

💡 Usa if/else if para cada operación.
O un objeto como mapa: const ops = {'+':sumar, '-':restar, '*':multiplicar, '/':dividir}
Luego: const r = ops[op](a,b)
