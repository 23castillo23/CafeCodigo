Variables — Las cajas con nombre

const vs let

📌 Crea una variable con let llamada puntos que empiece en 0. Luego cámbiala a 10. Luego cámbiala a 25. Muestra el valor final con console.log().

const no cambia — let sí cambia:

let contador = 0 contador = 5 ← funciona con let contador = 10 ← y otra vez console.log(contador) // 10

4

Unir textos

📌 Crea variables para nombre y apellido. Luego crea una tercera variable nombreCompleto que las una con un espacio en medio. Muéstrala con console.log().

Unir textos con + o con template literals:

const saludo = "Hola" const quien = "mundo" // Forma 1 — con + const mensaje1 = saludo + " " + quien // Forma 2 — con template literal (más limpio) const mensaje2 = `${saludo} ${quien}` console.log(mensaje1) // "Hola mundo" console.log(mensaje2) // "Hola mundo"

Escribe tu código:

💡 Para el espacio en medio: nombre + " " + apellido
O con template literal: `${nombre} ${apellido}`
Los backticks (`) no son comillas normales — en el teclado están con Alt+96 o cerca del 1.
5

Operaciones con números

📌 Crea dos variables numéricas: precio (100) y descuento (15). Calcula el precioFinal restando el descuento al precio. Muestra un mensaje que diga: "El precio final es: 85"

Operaciones matemáticas en JavaScript:

const a = 10 const b = 3 console.log(a + b) // 13 — suma console.log(a - b) // 7 — resta console.log(a * b) // 30 — multiplicación console.log(a / b) // 3.33 — división console.log(a % b) // 1 — resto de la división

Escribe tu código:

💡 const precioFinal = precio - descuento
Para el mensaje puedes usar: "El precio final es: " + precioFinal
O con template literal: `El precio final es: ${precioFinal}`

typeof — saber qué tipo es

📌 Crea 4 variables: una con texto, una con número, una con booleano y una con null. Usa typeof para mostrar el tipo de cada una.

typeof dice de qué tipo es un valor:

const x = "hola" console.log(typeof x) // "string" const y = 42 console.log(typeof y) // "number" const z = true console.log(typeof z) // "boolean"

Escribe tu código:

💡 typeof null devuelve "object" — eso es un bug histórico de JavaScript que nunca se corrigió para no romper sitios viejos. No es error tuyo, es un comportamiento conocido del lenguaje.
7

Convertir tipos

📌 Tienes el texto "42" (número guardado como texto). Conviértelo a número real con Number() y súmale 8. El resultado debe ser 50, no "428".

Convertir entre tipos:

const textoNum = "10" const numero = Number(textoNum) // 10 (número real) // Sin convertir: console.log(textoNum + 5) // "105" ← concatena texto // Con conversión: console.log(numero + 5) // 15 ← suma real

Escribe tu código:

💡 Number("42") convierte el texto "42" al número 42.
También existe parseInt("42") para enteros y parseFloat("3.14") para decimales.
El más común en la práctica es Number() o parseInt().
