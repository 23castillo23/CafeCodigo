Comparar con ===

Comparación estricta:

// === compara valor Y tipo exactos "hola" === "hola" ← true "hola" === "Hola" ← false (mayúsculas importan) 5 === "5" ← false (número vs texto) 5 === 5 ← true

💡 JavaScript distingue mayúsculas de minúsculas en los textos.
"sí" !== "Sí" !== "SI" — son tres textos diferentes.
Para comparar sin importar mayúsculas: respuesta.toLowerCase() === "sí"

else if — múltiples opciones

📌 Crea una variable nota con un número del 0 al 100. Muestra la calificación correspondiente:
• 90 o más → "Excelente"
• 80 o más → "Muy bien"
• 70 o más → "Bien"
• 60 o más → "Suficiente"
• Menos de 60 → "Reprobado"

else if para múltiples condiciones:

if (condicion1) { // primera opción } else if (condicion2) { // segunda opción } else if (condicion3) { // tercera opción } else { // si ninguna se cumplió }

Escribe tu código:

💡 El orden importa — pon las condiciones más restrictivas primero.
Si pones nota >= 60 primero, una nota de 95 entrará ahí y nunca llegará a "Excelente".
Prueba con diferentes valores: 95, 85, 72, 63, 45.

Operadores && y ||

📌 Crea variables tieneDinero (true) y estaAbierto (false). Muestra "Puede comprar" solo si ambas son true. Luego cambia estaAbierto a true y vuelve a ejecutar.

&& (Y) y || (O):

// && — ambas deben ser true if (condA && condB) { /* solo si las dos son true */ } // || — al menos una debe ser true if (condA || condB) { /* si una o las dos son true */ } // ! — invierte el booleano if (!condA) { /* si condA es FALSE */ }

Escribe tu código:

💡 Con &&: si cualquiera es false, el resultado es false.
Con ||: si cualquiera es true, el resultado es true.
Prueba cambiando los valores y predice el resultado antes de ejecutar.

Operador ternario

📌 El operador ternario es un if/else en una sola línea. Úsalo para crear una variable mensaje que sea "Par" si un número es par, o "Impar" si no. Muéstralo con console.log.

Operador ternario — if/else en una línea:

// condicion ? "si es true" : "si es false" const edad = 20 const tipo = edad >= 18 ? "adulto" : "menor" console.log(tipo) // "adulto" // Es lo mismo que: let tipo2 if (edad >= 18) { tipo2 = "adulto" } else { tipo2 = "menor" }

Escribe tu código:

💡 numero % 2 devuelve el resto de dividir entre 2.
Si el resto es 0 → par. Si es 1 → impar.
Prueba con 4, 7, 0, 100, 13.

Estructura del switch:

switch (variable) { case valor1: console.log("opción 1") break case valor2: console.log("opción 2") break default: console.log("ninguna coincidió") }

Escribe tu código:

El default es opcional pero buena práctica — maneja valores inesperados.
