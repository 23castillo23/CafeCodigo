🌐 HTML + CSS + JavaScript — el trío de la web

Todo lo que ves en un navegador está construido con estos tres lenguajes. Cada uno tiene un rol distinto y trabajan juntos.

🏗 HTML — estructura
Define el contenido y la estructura: párrafos, títulos, imágenes, listas. Es el esqueleto de la página.
🎨 CSS — estilo
Controla colores, tamaños, fuentes, espaciado y animaciones. Le da apariencia visual al HTML.
⚡ JavaScript — comportamiento
Añade interactividad: responde a clics, muestra/oculta elementos, valida formularios.
🖥 ¿Dónde se ejecuta?
Directamente en el navegador. No necesita instalación ni compilación.
