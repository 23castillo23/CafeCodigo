⚡ Consola JavaScript

Escribe JS y ve los resultados al instante. Usa console.log(), console.error(), console.warn() y más. Atajo: Ctrl+Enter para ejecutar.

⚡ JavaScript puro — la lógica del navegador

Ejecuta JavaScript sin HTML ni CSS, igual que en la consola del navegador. Perfecto para practicar lógica, funciones, arrays y objetos.

🖥 Consola del navegador (F12)
Presiona F12 en Chrome o Firefox → pestaña Console. Puedes escribir JS directamente ahí.
📋 console.log() — tu mejor amigo
Variantes: console.error() (rojo), console.warn() (amarillo), console.info() (azul).
⚡ Motor JS
El navegador tiene un motor JS integrado (V8 en Chrome). Node.js permite correr JS fuera del navegador.
🛠 Herramientas recomendadas
VS Code + Quokka.js · Chrome DevTools · Node.js en terminal
