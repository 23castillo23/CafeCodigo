🛠 ¿Qué consola o programa uso para cada lenguaje?

Una guía práctica de herramientas reales: dónde escribes, cómo ejecutas y dónde ves el resultado según el lenguaje.

📊 Comparativa rápida
Lenguaje ¿Dónde se ejecuta? ¿Necesita instalar? Consola de salida IDE recomendado
HTML + CSS Navegador (Chrome, Firefox…) ✗ No La propia página web VS Code + Live Preview
JavaScript Navegador ó Node.js ✗ No (navegador) DevTools → Console (F12) VS Code + Quokka.js
Python Intérprete Python / Pyodide ✓ Sí (opcional) Terminal / IDLE / Jupyter VS Code + ext. Python
Java JVM (Java Virtual Machine) ✓ Sí (JDK) Terminal (javac + java) IntelliJ IDEA / Eclipse
🔍 Detalle por lenguaje
🌐 HTML + CSS

Se ejecutan directamente en el navegador. No hay consola de salida visible; el resultado es la página que ves.

Local
VS Code + Live Preview
Instala la extensión "Live Preview" de Microsoft. Guarda el archivo .html y ves cambios en tiempo real en el editor.
Local
Abrir con el navegador
Doble clic al archivo .html → se abre en Chrome/Firefox. Sin instalación extra.
Online
CodePen / JSFiddle
Editores HTML+CSS+JS en el navegador. Perfectos para practicar sin instalar nada. codepen.io ↗
⚡ JavaScript

Corre en el navegador (cliente) o en Node.js (servidor/terminal). La consola es donde ves el output.

F12
DevTools del navegador
Presiona F12 → pestaña Console. Escribe JS directo y ves el resultado. Es la herramienta #1 del desarrollador web.
Local
Node.js en terminal
Instala Node.js → en terminal escribe node archivo.js. Los console.log() aparecen en la terminal.
IDE
VS Code + Quokka.js
Extensión que muestra los resultados de console.log directamente en el editor, línea por línea en tiempo real.
🐍 Python

Lenguaje interpretado. No necesita compilar: el intérprete lee y ejecuta el código línea a línea.

Terminal
Intérprete Python (REPL)
Escribe python en la terminal → entras al modo interactivo. python archivo.py ejecuta un archivo completo.
IDE
PyCharm / VS Code
PyCharm (JetBrains) es el IDE más completo para Python. VS Code + extensión Python también es excelente y más ligero.
Online
Google Colab / Jupyter
Para ciencia de datos e IA. Celdas de código interactivas en el navegador. colab.google.com ↗
☕ Java

Lenguaje compilado. El código .java se convierte a bytecode que ejecuta la JVM. Necesitas instalar el JDK.

Terminal
javac + java (terminal)
Instala JDK → javac Main.java compila → java Main ejecuta. El System.out.println() aparece en la terminal.
IDE
IntelliJ IDEA
El IDE más usado por profesionales de Java. Tiene versión Community gratuita. Incluye compilador integrado, depurador y autocompletado inteligente.
Online
OneCompiler / Replit
Sin instalar nada. Escribe y ejecuta Java en el navegador. onecompiler.com/java ↗
💡 El truco que todo dev conoce
🔍

Presiona F12 en cualquier página web

En Chrome o Firefox, F12 abre las Herramientas de Desarrollador. En la pestaña Console puedes escribir JavaScript directamente. También puedes inspeccionar cualquier elemento de la página, ver el HTML, CSS, solicitudes de red y errores. Es tu laboratorio secreto para aprender cómo funciona cualquier web.
