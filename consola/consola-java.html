☕ Java — lenguaje compilado de propósito general

Java no corre directamente en el navegador. Necesita un compilador que convierte tu código a bytecode, que luego ejecuta la JVM.

⚙ El ciclo de Java
Escribes código en .java → el compilador (javac) lo convierte a .class → la JVM lo ejecuta.
🖥 IDEs más usados
IntelliJ IDEA — el más popular · Eclipse — gratis y completo · VS Code + Extension Pack for Java
💻 En terminal (local)
javac Main.java compila y java Main ejecuta. La terminal muestra el output de System.out.println().
🌐 Online (sin instalar)
Para practicar rápido: OneCompiler, Replit, JDoodle. Presiona el botón arriba para abrir tu código directo.
⚠️

Java requiere un compilador

A diferencia de HTML, CSS y JavaScript que el navegador ejecuta directamente, Java necesita compilarse antes de correr. Por eso abrimos editores online especializados — el proceso es: escribes aquí → copias → pegas allá → ejecutas.
