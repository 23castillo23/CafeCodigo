🌐 HTML + CSS + JavaScript — el trío de la web

Todo lo que ves en un navegador está construido con estos tres lenguajes. Cada uno tiene un rol distinto y trabajan juntos.

🏗 HTML — estructura
Define el contenido y la estructura de la página: párrafos, títulos, imágenes, listas. Es el esqueleto.

Herramienta: cualquier editor de texto. El navegador lo interpreta directamente.
🎨 CSS — estilo
Controla colores, tamaños, fuentes, espaciados y animaciones. Le da apariencia visual al HTML.

Herramienta: VS Code + extensión Live Preview, o directamente aquí.
⚡ JavaScript — comportamiento
Añade interactividad: responder a clics, mostrar/ocultar elementos, validar formularios, hacer peticiones a APIs.

Herramienta: Consola del navegador (F12 → Consola) o VS Code.
🖥 ¿Dónde se ejecuta?
Directamente en el navegador (Chrome, Firefox, Safari). No necesita instalación ni compilación. El navegador es el motor que corre los tres a la vez.

⚡ Consola JavaScript

Escribe JS y ve los resultados en la consola inferior. Usa console.log(), console.error() y más.

⚡ JavaScript puro — la lógica del navegador

En esta pestaña puedes ejecutar JavaScript sin HTML ni CSS, igual que lo harías en la consola del navegador. Perfecto para practicar lógica, funciones, arrays y objetos.

🖥 Consola del navegador (F12)
Presiona F12 en Chrome o Firefox → pestaña Console. Puedes escribir JS directamente ahí. Es la herramienta que usan los devs todos los días.
📋 console.log() — tu mejor amigo
Muestra valores en la consola. Hay variantes: console.error() (rojo), console.warn() (amarillo), console.info() (azul). Úsalos para depurar.
⚡ ¿Cómo se ejecuta JS?
El navegador tiene un motor JS integrado (V8 en Chrome, SpiderMonkey en Firefox). También Node.js te permite correr JS fuera del navegador, en tu computadora.
🛠 Herramientas recomendadas
VS Code + extensión Quokka.js (muestra resultados en línea) · Chrome DevTools · Node.js en terminal · CodePen online

☕ Java — lenguaje compilado de propósito general

Java no corre directamente en el navegador. Necesita un compilador que convierte tu código a bytecode, que luego ejecuta la JVM (Java Virtual Machine).

⚙ El ciclo de Java
Escribes código en .java → el compilador (javac) lo convierte a .class (bytecode) → la JVM lo ejecuta. Por eso funciona en cualquier sistema operativo.
🖥 IDE más usados
IntelliJ IDEA — el más popular entre profesionales · Eclipse — gratis y muy completo · VS Code + Extension Pack for Java · NetBeans
💻 En terminal (local)
Con Java instalado: javac Main.java compila y java Main ejecuta. La terminal es la "consola" donde ves el output de System.out.println().
🌐 Online (sin instalar)
Para practicar rápido: OneCompiler, Replit, JDoodle. Presiona el botón arriba para abrir tu código directo en OneCompiler.
⚠️

Java requiere un compilador

A diferencia de HTML, CSS y JavaScript que el navegador ejecuta directamente, Java necesita compilarse antes de correr. Por eso lo practicamos en editores online especializados.

🐍 Python — sintaxis limpia, ideal para aprender lógica

Python es uno de los lenguajes más usados del mundo: ciencia de datos, IA, automatización, backend. Su sintaxis es muy legible, sin llaves ni punto y coma obligatorio.

⚙ ¿Cómo se ejecuta?
Python es interpretado: el intérprete lee y ejecuta línea por línea, sin compilar primero. Aquí usamos Pyodide, que corre Python real dentro del navegador vía WebAssembly.
🖥 Localmente (tu computadora)
Instala Python desde python.org. Luego en terminal: python archivo.py o python3 archivo.py. La terminal es tu consola de salida.
🛠 IDEs y editores
VS Code + extensión Python (Microsoft) · PyCharm (muy completo) · Thonny (ideal para principiantes) · Jupyter Notebook (para datos)
🌐 Online
Esta misma consola (Pyodide) · Google Colab (Jupyter gratis) · Replit · Python Tutor (visualiza el código paso a paso)

Corre en el navegador (cliente) o en Node.js (servidor/terminal). La consola es donde ves el output.

Presiona F12 → pestaña Console. Escribe JS directo y ves el resultado. Es la herramienta #1 del desarrollador web.

☕ Java

Lenguaje compilado. El código .java se convierte a bytecode que ejecuta la JVM. Necesitas instalar el JDK.

Terminal
javac + java (terminal)
Instala JDK → javac Main.java compila → java Main ejecuta. El System.out.println() aparece en la terminal.
IDE
IntelliJ IDEA
El IDE más usado por profesionales de Java. Tiene versión Community gratuita. Incluye compilador integrado, depurador y autocompletado inteligente.
Online
OneCompiler / Replit
Sin instalar nada. Escribe y ejecuta Java en el navegador. onecompiler.com/java ↗
💡 El truco que todo dev conoce — la consola F12
🔍

Presiona F12 en cualquier página web

En Chrome o Firefox, F12 abre las Herramientas de Desarrollador. En la pestaña Console puedes escribir JavaScript directamente. También puedes inspeccionar cualquier elemento de la página, ver el HTML, CSS, solicitudes de red y errores. Es tu laboratorio secreto para aprender cómo funciona cualquier web.
