HTML

Los bloques de construcción

Etiquetas, elementos y atributos — el vocabulario base de toda página web.

🏷️

Etiqueta

El ladrillo básico de HTML

🧱 Como un ladrillo de construcción — define qué tipo de cosa es, no cómo se ve.

Es la instrucción básica de HTML. Le dice al navegador qué tipo de elemento existe: un botón, un título, una imagen, una caja. Siempre tiene apertura y cierre.

HTML 
<button> Nuevo Grupo </button>
<h1> FotoApuntes </h1>
<p> Álbum Escolar </p>
<div> una caja invisible </div>
<!-- div agrupa cosas sin forma ni color propio -->
💡

En tu proyecto FotoApuntes: button, h1, h2, p, div, input, section — todas son etiquetas de tu index.html.

🪆

Elemento

La cosa completa

🪆 Como una muñeca rusa completa — etiqueta apertura + contenido + etiqueta cierre juntos.

La etiqueta es solo el nombre. El elemento es la cosa entera.

HTML 
<!-- Esto es UN elemento completo -->
<button class="btn-new-group" id="btnNewGroup">
  Nuevo Grupo
</button>

↑ etiqueta apertura + atributos
  ↑ contenido
    ↑ etiqueta de cierre
Diferencia clave: La etiqueta es solo <button> — el nombre. El elemento es todo: apertura + contenido + cierre.
📋

Atributo

Datos extra que describen el elemento

🏷️ Como las características de un producto — color: rojo, talla: M, precio: $50. Van dentro de la etiqueta de apertura con la forma nombre="valor".

Un elemento puede tener muchos atributos a la vez. Dos de ellos son especiales: class y id.

HTML 
<button class="btn-new-group" id="btnNewGroup">
         ↑ atributo 1          ↑ atributo 2

<input type="text" placeholder="Escribe aquí...">
<img   src="foto.jpg"  alt="Mi foto">

CSS class

Para dar estilo. Se repite en muchos elementos. En CSS se escribe con punto: .btn-new-group

JS id

Para que JS encuentre ese elemento. Único en toda la página. Se usa: getElementById('...')

💡

En tu proyecto: class, id, type, placeholder, href, src, accept — todos son atributos de tu index.html.

🏠

Anidamiento

Elementos dentro de elementos

🏠 Como una casa — la casa contiene cuartos, los cuartos contienen muebles. El de afuera es el padre, el de adentro es el hijo.

El anidamiento describe dónde vive cada elemento. Elemento es ¿qué es? — Anidamiento es ¿dónde está?

HTML 
<header>                            ← abuelo
  <div class="header-inner">         ← padre
    <div class="header-icon">📸</div>  ← hijo 1
    <h1>FotoApuntes</h1>             ← hijo 2
    <p>Álbum Escolar</p>              ← hijo 3
  </div>
</header>
header
  └── div.header-inner        ← padre
        ├── div.header-icon   ← hijo 1
        ├── h1                ← hijo 2
        └── p                 ← hijo 3
💡

En tu proyecto: toda la página es anidamiento — secciones dentro de main, botones dentro de divs, modales dentro del body.
